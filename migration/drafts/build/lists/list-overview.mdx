## What is a list?

A list is a single-column [dataset](/t/p8h4alk/dataset-and-views-overview). Forms in your flows can use the list data in dropdowns, checklists, checkbox, or multi-select fields.

### Common list examples

-   Department.
-   Leave type.
-   Countries.
-   States.
-   Bank account type.
-   Payment type.![](/migration-assets/list-overview/1.png)

## Where to use lists in Kissflow

### [Dropdown field](/t/35h4azp/dropdown-multi-select-dropdown-checkbox-checklists-and-radio-buttons#dropdown-field)

Use when a single option has to be selected from a list of options. For example, on a paid time off (PTO) process, users can select a single type of leave, such as sick, vacation, personal, and so on.

### [Checklist field](/t/35h4azp/dropdown-multi-select-dropdown-checkbox-checklists-and-radio-buttons#checklist-field)

You can add a checklist of items that need to be completed as part of a workflow step. For example, some steps in an employee onboarding process might include checklists like IT creating accounts for multiple software applications.

### [Checkbox or multi-select field](/t/35h4azp/dropdown-multi-select-dropdown-checkbox-checklists-and-radio-buttons#multi-select-dropdown)

Use it when you want to let users select multiple items. For example, in a travel request process, users can select whether they need air, local, and/or accommodation assistance.

> **Note:**
> 
> You can add up to 250 list items, each with a limit of 500 characters.
