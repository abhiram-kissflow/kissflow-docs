We have classified the fields into two primary categories: **All fields** and **AI-suggested** fields for your easy understanding. The **All fields** tab is further organized into sub-categories such as:

-   Basic fields.
    
-   Advanced fields.
    
-   File and media.
    
-   Data lookup.
    
-   Widget.
    

![](/migration-assets/basic-and-advanced-fields/1.gif)

Navigate to the respective form editor in your flow to access the **Fields** panel on the left side. Toggle between the **All fields and AI-suggested** fields tabs to access the field that you would like to use in your flow form. You have to drag and drop a field from this panel onto your form editor to add it. You can also click the empty area in your form section to search for a field and add it instantly quickly.

## Basic fields

The following are the available basic fields:

**Field**

**Description**

[Text](/t/p8h4az5/text-text-area-email-and-rich-text-fields)

Stores a single line of text.

[Text area](/t/p8h4az5/text-text-area-email-and-rich-text-fields#text-area-field)

Stores multiple lines of text with limited formatting options.

[Number](/t/60h4az2/number-field#number-field)

Stores numbers or decimals.

[Date](/t/x2h4azf/date-and-date-time-field#date-field)

Stores date only.

[Date & Time](/t/x2h4azf/date-and-date-time-field#date-amp-time-field)

Stores date along with timestamp.

[Dropdown](/t/35h4azp/dropdown-checkbox-checklist#dropdown-field)

Stores multiple values but lets the user select only one value at a time.

[Yes/No](/t/35h4azn/yes-no-field)

Stores binary information like Yes or No.

[Multi-select dropdown](/t/35h4azp/dropdown-checkbox-checklist#multi-select-dropdown)

Stores multiple values and lets the user select more than one value at a time.

[Checkbox](/t/35h4azp/dropdown-checkbox-checklist#checkbox-field)

Stores multiple values and lets the user check one or more values at a time.

[Currency](/t/m1h4azb/currency-field)

Stores currency values.

[Email](/t/p8h4az5/text-text-area-email-and-rich-text-fields#email-field)

Stores valid email addresses.

[User](/t/h7h4azj/user-field)

Stores usernames only.

[Radio button](/t/35h4azp/dropdown-multi-select-dropdown-checkbox-checklists-and-radio-buttons#radio-button)

Stores multiple values and lets users select only one option at a time.

## Advanced fields

The following are the available advanced fields:

**Field**

**Description**

[Aggregation](/t/83h4azk/aggregation-field)

Collects data from different fields and lets users perform operations like SUM, AVERAGE, CONCATENATE, and so on.

[Signature](/t/y4h4az1/image-attachment-and-signature#signature-field)

Stores the user's signature. The user can directly sign in the area provided or upload their signature file accordingly.

[Sequence number](/t/y4h4azw/sequence-number)

Generates a number in sequence number automatically whenever a flow item is submitted.

[Scanner](/t/83h4azd/scanner-field)

Scans QR codes and barcodes. 

[Geolocation](/t/q6h4az7/geolocation-field)

Store location of the users with their consent. 

## Widget

The following are the available widgets:

**Field**

**Description**

[Slider](/t/60h4az2/number-field#slider-field)

Lets the user slide over a number scale to feed their input. The value is stored as a number.

[Rich text](/t/p8h4az5/text-text-area-email-and-rich-text-fields#rich-textfield)

Stores multiple lines of text supported along with many formatting options.

[Rating](/t/60h4az2/number-field#rating-field)

Stores rating values.

[Checklist](/t/35h4azp/dropdown-checkbox-checklist#checklist-field)

Displays a list of options that the form user can check off.

[Grid](/t/q6yywc7/grid-widget)

Stores rows and columns to optimize space.

Button

Stores on-click event actions. 

## File & Media

The following is the list of fields that support attachments:

**Field**

**Description**

[Attachment](/t/y4h4az1/image-attachment-and-signature#attachment-field)

Stores many file types like DOC, PDF, XLS, XLSX, ODS, and so on.

[Image](/t/y4h4az1/image-attachment-and-signature#image-field)

Stores image files like JPG, JPEG, PNG.

[Smart attachment](/t/y4h4az1/image-attachment-and-signature#smart-attachment-field)

Fetches data from an attachment like a receipt or an invoice, thereby saving the end-user time manually filling in the form fields.

## Data lookup

The following is the list of lookup fields:

**Field**

**Description**

[Lookup](/t/g9h4azr/lookup-field)

Loads data from other flows onto the form.

[Remote lookup](https://helpdocs.kissflow.com/remote-lookup/extracting-a-text-and-number-using-remote-lookup)

Fetches data from an external data source.
