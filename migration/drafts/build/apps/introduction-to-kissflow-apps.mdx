## What is a Kissflow App?

Kissflow Apps uses a low-code app development approach that requires little to no coding in order to create applications. It uses visual tools and custom scripting to develop and deliver complex applications at scale.

You can utilize the Kissflow Apps framework to create dynamic user applications and workflows on top of Kissflow. You can use them to create custom dashboard views for Kissflow Processes and Boards, build tight integrations, or display layouts from reports and data forms in Kissflow.

Based on specified user roles, users can then interact with these apps inside their Kissflow Work Platform. Your apps can be purely client-side apps or make requests to remote resources.

> **Note:** Please note that the App builder is available only for the **Kissflow Work Platform Enterprise** plan.

### How's it different from traditional application development?

Rather than using traditional coding languages such as Java, C++, Python, etc., Kissflow's app platform has drag-and-drop features and pre-configured components that allow for application development with little or no coding. You can also use various flavors of JavaScript to add custom features to your application. This not only makes application development easier and more accessible to non-professional developers but also speeds up the overall development significantly.

Apart from your primary Kissflow account, you'll also have access to separate development and test environments to build and test your applications securely.

### When to use Kissflow Apps?

Applications play an important role in the automation of workflows. Processes in Kissflow allow you to deploy and run small process flows in no time. This is very convenient if you need to get started quickly and you have a very limited set of people using the flow. However, if your use case necessitates a comprehensive application and you have a large number of stakeholders, developing a low-code application is the best option.

For instance, each tool used in workflows such as process management, case management, or document management is an application. They are usually hosted on a platform such as BPM, CRM, or ERP software. Additionally, they connect with third-party software, such as Google Sheet, Gmail, Slack, etc., using integrations. If your business falls in any of the above categories, you should consider using the Kissflow Application builder to create an application.

With Kissflow Apps, businesses can create customizable applications and allow non-professional developers to create apps in a very short span of time. Anyone in a company, from HR managers to Admin staff to sales representatives, can develop new applications in a relatively short period of time and without the need for professional coding knowledge. This enables the company to speed-up application rollout, accelerating digital transformation and optimizing business processes to promote efficiency and growth.
