In addition to the existing formula builder, the AI add-on simplifies computing and managing formulas. Enterprise plan users can effortlessly incorporate computed fields using prompts. 

> **Note:**
> 
> The AI formula builder does not support Integrations.

## Generating formulas using prompts

To add a formula to a field using prompts:

1.  Navigate to the field’s settings and enable the **Is this a computed field?** toggle button.
    
2.  In the AI-powered space within the formula builder, enter your prompt and click the **Generate** button. The formula will be automatically generated and displayed in the builder.
    
3.  Review the formula and click **Save** to apply it.
    

![](/migration-assets/ai-formula-builder/3.gif)

## Evaluating formulas using AI

Once you’ve generated the formula using AI or entered it manually, you can test it using **AI Evaluate**.

A list of fields with auto-filled data entries and results will appear in the AI evaluate popup. You can either use the auto-filled inputs or modify them as needed. If modifying the inputs, click **Test** under Results to verify the output.

![](/migration-assets/ai-formula-builder/1.gif)

For more information, refer to the [guide on input behaviors for different field types](/t/m1ylbzc/handling-input-behavior-for-various-fields-in-ai-evaluation).

## Generating description for formulas

If you have existing formulas or prefer to write them manually, you can view their descriptions in the AI-powered space.

To access the formula builder, click the formula preview below the **Is this a computed field?** toggle button. When the formula builder opens, the formula's description automatically loads in the AI space.

![](/migration-assets/ai-formula-builder/4.gif)

If you manually change an existing formula, you can refresh the prompt to generate an updated description.

![](/migration-assets/ai-formula-builder/2.gif)
