## What is Asana?

Asana is a web and mobile work management platform designed to help teams organize, track, and manage their work. To stay on top of project priorities, you can organize your tasks on a board, list, or timeline view.

## Connecting to Asana

The Asana connector of Kissflow Integration enables you to connect and synchronize your Asana projects and tasks with popular third-party apps and Kissflow apps. You must have a valid Asana account and be a paid Kissflow customer in order to integrate with Asana.Assume you run an email marketing campaign through MailChimp, and you want to perform certain actions on your new email subscribers. Connecting Asana and MailChimp allows you to create new tasks for your new email subscribers easily.To perform specific actions, use the Asana connector to receive data from other flows or apps. 

### Triggers in Asana connector

**Trigger events**

**Description**

When a new task is created

Triggers when a new task is created in an Asana project. 

When a task is updated

Triggers when an existing task is updated.

When a new subtask is created

Triggers when a new subtask is created in an existing task.

When a new milestone task is created

Triggers when a new milestone task is created in the project.

When a milestone task is updated

Triggers when a milestone task is updated in the Project.

### Actions in Asana connector

**Action events**

**Description**

Create a task

This action creates a new task in your Asana project.

Update a task

This action updates an existing task in your Asana project.

Create a subtask

This action creates a new subtask and adds it to the parent task in your Asana project.

Create a section in the project

This action creates a new section in your Asana project.

Create a tag 

This action creates a new tag in a workspace or organization. 

Create a story on a task

This action adds a new story to a task.

Get a task

This action returns the complete task record for a particular task.

## Asana connector as an action step

1.  In your Kissflow account, click the **Create** button in the left navigation panel and select **Integration** from the list. Enter a unique name and description for your integration and click **[Create](/t/60h4akx/creating-an-integration)**. 
    
2.  Set up trigger step, for example, it can be **When an item is created**. Each time a new item is created in your Kissflow Process, this can lead to an action in the integration.
3.  After setting up your trigger step above, click the **Add** button (![](/migration-assets/xero-connector/1.png)) next to set up your Asana action step. Search for **Asana** and select an available action event. ![](/migration-assets/asana-connector/1.png)
4.  After choosing your action event, next step is to authenticate your connector. Click **+Add an account** and sign in to your Asana account. You will be prompted to use OAuth to authenticate your Asana account.
5.  After authenticating with your Asana account credentials, add the relevant values against the Asana form fields, hit **Refresh fields** to view all the updated fields in the selected Asana form. You can also map Asana fields with your trigger step or previous action's fields. After the action is executed, the field values will be automatically added to the task fields.![](/migration-assets/asana-connector/2.png)
6.  After a successful configuration, test whether your Asana action step is set up correctly by clicking the **T****est** button. The connector will pull field data from your selected Asana account and display it as a JSON output. Similar to the trigger step, errors are notified to you right away. Fix them before retesting the action step.

## Setting up an Asana workflow

After creating a custom workflow with your Asana action step, switch **ON** the toggle button to activate the integration run. When the integration is active, a task will be newly created or updated based on a trigger event you had set.

To ensure your integrations are properly configured and tested, review our [best practices guide](/t/p8yzsxn/best-practices-for-using-kissflow-integrations) for tips on building robust and maintainable solutions.
