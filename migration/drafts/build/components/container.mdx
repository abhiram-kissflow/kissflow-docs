#### Container

A container is a tool used to build a set of components within a defined space. Within a container, you will be able to add different components that are available within the page builder. You can configure two types of page layouts for container: [Flex and Grid](/t/m1ykbax/working-with-page-layouts).

![](/migration-assets/container/1.png)

#### How to use it?

Containers can be used as the body of that defined space. It allows you to add multiple components, such as [buttons](/t/x2yzj42/button), [labels](/t/g9yzj40/label), etc., within that container.

Each container has a unique container id specific to that container. 

**The following actions can be performed under** [**Style properties**](/t/y4yzj9m/managing-component-style-properties) **in the container:** 

1.  Modify the alignment.
2.  Set the sizing.
3.  Adjust the padding and margin.
4.  Set border size, color, and radius.
5.  Apply shadow and background color.

> **Note:** Events cannot be triggered in a container.
