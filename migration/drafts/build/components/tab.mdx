#### Introduction

A tab component lets you categorize content into different tabs. Using a tab container, you can create a horizontal navigation bar with individual tabs having a container(link) of its own. It basically lets you create a secondary navigation for your application.

![](/migration-assets/tab/1.gif)

#### How to use it?

Once you drag and drop the Tab component, you will be able to perform the following action:

1.  Choose a tab style for the tab.
2.  Under Tab settings, you can:

-   Edit the tab name.
-   Make a particular tab as the default tab.
-   Add additional tabs.

1.  Set the color, active color, and hover color for the tab.
2.  In Style properties, you will be able to adjust the sizing, modify the padding and margin, and set the border size, color, and radius.
3.  In Event properties, you will be able to add the event “On Tab change” to direct the tab to perform an action. 

**There are three actions that can be performed:**

1. **Redirection** - This action redirects you to the user-defined home page, the system-defined home page, or lets you go back to the same page.
2. **Open popup** - The open popup action lets you open a popup that you had created using the [Popup](/t/35yzj4w/popup) component.
3. **JavaScript action** - The JavaScript action lets you define the action that you want to perform using JS code.
