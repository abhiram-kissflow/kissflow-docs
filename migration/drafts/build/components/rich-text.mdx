## What is rich text?

The **Rich text** component allows you to add styled text content directly to your page. It supports headings, paragraphs, links, lists, images, and basic formatting (like bold, italic, and underline), making it ideal for adding informational or descriptive content without writing any code.

You can use the rich text component to insert instructions, descriptions, labels, or any other static content that enhances your page layout. It’s fully customizable and can be styled to match the look and feel of your app.

## How to use it?

1.  Drag and drop the rich text component onto your page.
    
2.  Click inside the component to type in your text directly into the built-in text editor.
    
3.  Type a forward slash / to get the commands drop-down. Here, you can find formatting and styling options.
    
4.  Use the **Fields** option in the commands drop-down to insert variables and parameters.
    

### Using commands for rich text

The commands dropdown has the following options:

-   **Inline formatting**: Bold, italic, underline, strikethrough, subscript, superscript.
    
-   **Text structure**: Headings (H1, H2), paragraph, block quote.
    
-   **Lists**: Bulleted and numbered list.
    
-   **Links and media**: Add hyperlinks, images, or embedded content.
    
-   **Code snippet**: Provides you with a code block to add a code snippet to your page.
    
-   **Variable support**: The **Fields** option helps you insert dynamic values using local or global variables.

> **Note:** For security reasons, the rich text component treats the code snippets pasted into it as plain text and does not interpret them as code.

### Visibility

In the **Visibility** tab, you can set a default visibility and add visibility rules for the rich text component, if you wish to show or hide it under certain conditions.

### Style properties

Under [Style properties,](/t/y4yzj9m/managing-component-style-properties) you can choose from a variety of formatting options for the rich text component. You will be able to adjust the sizing, modify the padding and margin, and set the border size, color, and radius and this will be applied to the component as a whole.

Select the text within the editor to invoke the formatting toolbar to apply formatting exclusively to the text within the rich text editor.
