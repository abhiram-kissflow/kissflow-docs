#### Introduction

A label can be used to add a specific text label for your page. For instance, you may have different sections in your page and you can use different text labels to identify these sections.

![](/migration-assets/label/1.png)

#### How to use it?

Once you drag and drop the Label component, you will be able to perform the following actions:

1.  Edit the preexisting label text to a text of your wish.
2.  Set the font size, color, background color, and text style such as Bold, Italics, Underline, and Strikethrough.

In [Style properties,](/t/y4yzj9m/managing-component-style-properties) you will be able to adjust the sizing, modify the padding and margin, and set the border size, color, and radius.
