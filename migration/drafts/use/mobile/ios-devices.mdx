If you are using an iOS device, you can use your browser to access your Kissflow account or a Progressive Web App to access it.

## Using a mobile browser

Use your preferred mobile browser to navigate to the URL where your account is located. It will be **&lt;accountname>.kissflow.com.**

## Using a Progressive Web App (PWA)

A Progressive Web App gives a native app-like user experience and functions just like a mobile app. It also consumes less data and does not require any updates. 

![](/migration-assets/ios-devices/1.jpg)

> **Note:**
> 
> Minimum system requirements: 2GB RAM

## Installing PWA

1.  Open Safari.
    
2.  Type the account URL on the address bar. (**https://&lt;domainname>.kissflow.com**).
    
3.  Tap **Actions.**
    
4.  Tap **Add to Home Screen.**
    
5.  Tap **Add.**

> **Warning:**
> 
> iOS prevents Chrome users from adding a PWA to the home screen.
