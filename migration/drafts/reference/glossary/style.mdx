Style is a configuration that defines the visual appearance of a form and its various building blocks. This includes color schemes for [sections](/t/m1y3m5r/section-and-table-style#section-style), [tables](/t/m1y3m5r/section-and-table-style#table-style), and [fields](/t/60h4azc/field-appearance), as well as layout preferences for help text.
