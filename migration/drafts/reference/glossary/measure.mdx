In Analytics, a measure is an attribute used to compare data on a [chart](/t/q6h9qz5/pivot-reports). It is displayed on the Y-axis of a graphical chart. You can choose to display multiple measure values in a chart/pivot report. Measures are aggregated (Sum, Average, Count, Min., Max., Std Dev) to describe the data.
