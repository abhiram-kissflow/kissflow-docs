[Impersonation](/t/p8h4akb/impersonation-in-service-accounts) allows [service accounts](/t/g9h4akd/service-account-overview) to act on behalf of users or items within a Kissflow account. For example, it can make changes to user details, access all processes and integrations, or perform actions within specific flows.
