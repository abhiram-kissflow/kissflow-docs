[AI-suggested fields](/t/y4yqkrg/ai-suggested-fields) are a list of form fields automatically generated by AI. Unlike standard fields that require manual selection and configuration, these fields are suggested based on the provided form name and come pre-populated with a relevant Field name and Field type.
