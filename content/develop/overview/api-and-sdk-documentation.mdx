# API and SDK documentation

Kissflow's API reference and SDK documentation are maintained on the Kissflow developer portal, separate from this site. Use them to build integrations, automate actions, and extend Kissflow with custom code.

## API reference

The [API documentation](https://api.kissflow.com/) covers the REST APIs available for Kissflow apps, boards, processes, and other entities — including authentication, request and response formats, and available endpoints.

## SDK documentation

The [SDK documentation](https://developers.kissflow.com/gettingstarted/) covers the Kissflow SDK, used to build custom components and extend the product's UI and behavior in JavaScript.
