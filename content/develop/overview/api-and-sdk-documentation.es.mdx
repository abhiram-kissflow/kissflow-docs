# Documentación de API y SDK

<PlanBadge plans={["enterprise"]} />

La referencia de API y la documentación del SDK de Kissflow se mantienen en el portal para desarrolladores de Kissflow, separado de este sitio. Úsalas para crear integraciones, automatizar acciones y ampliar Kissflow con código personalizado.

## Referencia de API

La [documentación de API](https://api.kissflow.com/) abarca las API REST disponibles para las aplicaciones, los tableros, los procesos y otras entidades de Kissflow, incluidos la autenticación, los formatos de solicitud y respuesta, y los endpoints disponibles.

## Documentación del SDK

La [documentación del SDK](https://developers.kissflow.com/gettingstarted/) abarca el SDK de Kissflow, que se utiliza para crear componentes personalizados y ampliar la interfaz de usuario y el comportamiento del producto en JavaScript.

El [repositorio del SDK de JavaScript de Kissflow](https://github.com/kissflow/lcnc-sdk-js) contiene el código fuente del paquete y recursos relacionados con el desarrollo.
