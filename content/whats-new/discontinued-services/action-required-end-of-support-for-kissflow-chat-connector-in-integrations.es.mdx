# Acción requerida: fin de la compatibilidad con el conector de Kissflow Chat en Integrations

A partir del **20 de diciembre de 2023**, Kissflow Integrations dejó de admitir el conector de Kissflow Chat. Cualquier integración que dependa de este conector dejó de funcionar después de esa fecha.

## Qué hacer

Si tienes integraciones creadas con el conector de Kissflow Chat, cámbialas a otra plataforma de chat compatible. Actualmente, Kissflow Integrations admite conectores para Google Chat, Microsoft Teams y Slack; vuelve a configurar tu integración para usar una de estas opciones.

Si necesitas ayuda con la transición, [contacta con el soporte de Kissflow](/docs/whats-new/important-notices/slas-and-support-models#how-to-reach-support).
