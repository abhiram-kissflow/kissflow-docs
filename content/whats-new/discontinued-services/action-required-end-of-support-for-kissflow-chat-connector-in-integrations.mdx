# Action required: End of support for Kissflow Chat connector in Integrations

Effective **20 December 2023**, Kissflow Integrations no longer supports the Kissflow Chat connector. Any integration that depends on this connector stopped working after that date.

## What to do

If you have integrations built on the Kissflow Chat connector, switch them to another supported chat platform. Kissflow Integrations currently supports connectors for Google Chat, Microsoft Teams, and Slack — reconfigure your integration to use one of these instead.

If you need help with the transition, contact Kissflow support.
