La interfaz de Kissflow está disponible en los siguientes 20 idiomas.

| Idioma | Nombre nativo |
| --- | --- |
| Inglés | English |
| Chino simplificado | 简体中文 |
| Italiano | Italiano |
| Coreano | 한국어 |
| Japonés | 日本語 |
| Español | Español |
| Portugués brasileño | Português (Brazil) |
| Francés | Français |
| Alemán | Deutsch |
| Ruso | Русский |
| Vietnamita | Tiếng Việt |
| Tailandés | ภาษาไทย |
| Árabe | العربية |

| Hindi | हिन्दी |
| Rumano | Română |
| Persa | فارسی |

<Callout type="info">
  El árabe solo está disponible si se ha habilitado para tu cuenta. [Contacta con el equipo de soporte de Kissflow](/docs/whats-new/important-notices/slas-and-support-models#how-to-reach-support) si necesitas activar el árabe para tu cuenta.
</Callout>

Todos los demás idiomas de la tabla están disponibles para todas las cuentas de forma predeterminada; no se requiere ninguna configuración adicional para seleccionarlos.

## Cambiar el idioma de la interfaz

Tu preferencia de idioma personal anula el idioma predeterminado de tu organización, que los administradores establecen para toda la cuenta. Consulta [Información personal, configuración de formato y delegación de tareas](/docs/use/personal-settings/personal-info-format-settings-and-task-delegation) para saber cómo cambiar tu propia configuración.
