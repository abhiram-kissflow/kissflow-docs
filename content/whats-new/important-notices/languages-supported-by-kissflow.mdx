Kissflow's interface is available in the following 20 languages.

| Language | Native name |
| --- | --- |
| English | English |
| Simplified Chinese | 简体中文 |
| Italian | Italiano |
| Korean | 한국어 |
| Japanese | 日本語 |
| Spanish | Español |
| Brazilian Portuguese | Português (Brazil) |
| French | Français |
| German | Deutsch |
| Russian | Русский |
| Vietnamese | Tiếng Việt |
| Thai | ภาษาไทย |
| Arabic | العربية |
| Polish | Język Polski |
| Slovak | Slovenčina |
| Czech | Čeština |
| Ukrainian | Українська |
| Hindi | हिन्दी |
| Romanian | Română |
| Persian | فارسی |

<Callout type="info">
  Arabic is only available if your account has been enabled for it. Contact Kissflow support if you need Arabic turned on for your account.
</Callout>

Every other language in the table is available to all accounts by default — no additional setup is required to select it.

## Changing your interface language

Your personal language preference overrides your organization's default language, which admins set account-wide. See [Personal info, format settings, and task delegation](/docs/use/personal-settings/personal-info-format-settings-and-task-delegation) for how to change your own setting.
