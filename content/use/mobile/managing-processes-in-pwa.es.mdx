# Administrar procesos en la PWA

El panel de Kissflow Progressive Web App (PWA) tiene dos widgets para trabajar con elementos de procesos: **Mis tareas** y **Mis elementos**.

## Mis tareas

Una tarea es un elemento que está esperando tu aprobación u otra acción. El widget **Mis tareas** de la página **Inicio** muestra tus tareas en curso y atrasadas, y puedes añadir filtros según la fecha de vencimiento. Toca **Mostrar todo** en la sección **Mis tareas** para abrir una lista detallada de todas tus tareas.

### Administrar aprobaciones

1. Toca el widget **Mis tareas** en el panel.
2. Toca una tarea de la lista.
3. Toca **Aprobar** para aprobar el elemento.
4. Toca **Más acciones** para realizar otra acción. Desde aquí puedes rechazar, devolver o reasignar el elemento.

   <Callout type="info">
     Retirar no está disponible desde **Más acciones** en una tarea. Retirar un elemento solo está disponible para la persona que lo inició, desde el widget **Mis elementos**.
   </Callout>

### Otras acciones

1. Toca el widget **Mis tareas** en el panel.
2. Toca una tarjeta de tarea de la lista.
3. Toca el menú **Más opciones** en la esquina superior derecha de la tarea.
4. Toca **Progreso** para ver el flujo de trabajo y el estado actual de la tarea.
5. Toca el botón **Comentarios** (![Icono de comentarios](/migration-assets/managing-processes-in-pwa/2.png)) para dejar un comentario sobre la tarea.

   ![Vistas Progreso y Comentarios abiertas desde el menú Más opciones de una tarea](/migration-assets/managing-processes-in-pwa/3.gif)

<Callout type="info">
  Si todavía no tienes ninguna tarea asignada, puedes mantenerte al día [activando las notificaciones push](/docs/use/mobile/android-devices#enabling-push-notifications).
</Callout>

## Mis elementos

El widget **Mis elementos** muestra los elementos que has iniciado. Puedes filtrarlos por estado y período de creación.

1. Toca el widget **Mis elementos** en el panel.
2. Toca una tarjeta de elemento de la lista.
3. Toca **Retirar** o **Reasignar**, según lo que quieras hacer.
4. Toca el menú **Más opciones** en la esquina superior derecha del elemento para ver su progreso o duplicarlo.
