# Form child tables in the PWA and native mobile app

<PlanBadge plans={["basic", "enterprise"]} />

Kissflow's PWA and native [Android](/docs/use/mobile/android-devices) and [iOS](/docs/use/mobile/ios-devices) apps let you view and edit child tables inside process forms, board forms, and data forms. You can switch between two display modes depending on how you want to work with the data:

- **Tabular view** — the default view. Shows multiple rows at once as a table, making it easy to compare entries and edit several rows without expanding each one individually.
- **Accordion view** — keeps the child table compact by collapsing each row into its own expandable section, so you can focus on one row at a time.

## Switch between display modes

1. Sign in to your Kissflow account using a PWA or native mobile app.
2. Open any form that contains a child table.
3. Tap the **More options** button at the top of the child table.
4. Tap **Change display mode**.
5. Select **Tabular** or **Accordion**.

## Edit data in a child table

1. Tap any cell in the child table to open the editing modal.
2. Update the field values as needed. Use the previous and next buttons in the modal to move between rows without closing it.
3. Changes are saved automatically as you edit.

## View in fullscreen (native apps only)

<Callout type="info">
  Fullscreen view is only available on the native Android and iOS apps. It isn't available on the PWA.
</Callout>

Fullscreen view rotates the child table into landscape mode so you can see more columns at once — useful on devices with limited screen space.

1. Sign in to your Kissflow account using a native mobile app.
2. Open any form that contains a child table.
3. Tap the **More options** button at the top of the child table.
4. Tap **Enter fullscreen**.
5. When you're done, tap the **Exit fullscreen** button to rotate the screen back to portrait mode.

## Reorder columns

1. Sign in to your Kissflow account using a PWA or native mobile app.
2. Open any form that contains a child table.
3. Tap the **More options** button at the top of the child table.
4. Tap **Reorder columns**.
5. Press and hold the column header you want to move, then drag it to the desired position.
6. Release to drop the column in its new position, then tap **Save**.

<Callout type="info">
  Tap **Reset** at any point to restore the columns to their original order.
</Callout>

## Delete rows in bulk

You can select and delete multiple rows at once from the tabular view.

1. Sign in to your Kissflow account using a PWA or native mobile app.
2. Open any form that contains a child table.
3. In the tabular view, long-press any row to turn on row selection. A selection checkbox appears on the left of every row.
4. Tap the checkbox on each additional row you want to delete.

   ![Selection checkbox shown on the left of a row in the tabular view](/migration-assets/form-child-tables-in-the-pwa-and-native-mobile-app/3.png)
   {/* TODO: verify screenshot matches current product UI */}

5. Tap the **Delete** button at the top of the table.

   ![Delete button at the top of a child table with rows selected](/migration-assets/form-child-tables-in-the-pwa-and-native-mobile-app/4.png)
   {/* TODO: verify screenshot matches current product UI */}

6. Confirm the deletion to remove the selected rows.
