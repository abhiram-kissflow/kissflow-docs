### Overview

Kissflow's PWA and native [Android](/t/p8h9qzc/android-devices) and [iOS mobile app](/t/q6h9qz0/ios-devices) allow for flexible viewing and editing of child tables within forms. As a user of process form, board form, or data form, you can decide how to display the child table by switching between the default tabular view and an accordion view.

-   _Tabular view:_ The table format lets users view multiple entries simultaneously, making comparisons and edits more manageable without expanding each entry individually.
-   _Accordion view:_ This view keeps the child table layout clean and compact, providing users with a clear overview and the option to explore specific rows in detail.

### Switching between display modes

1.  Sign in to your Kissflow account using a PWA or native mobile app.
    
2.  Open any form containing a child table.
    
3.  Click the **More options** button (![](/migration-assets/form-child-tables-in-the-pwa-and-native-mobile-app/1.png)) at the top of the child table > click **Change** **display mode**.
    
4.  Select either **Tabular** or **Accordion** to switch views as needed.
    

### Fullscreen view

Switch to the landscape mode to view more columns simultaneously for a broader perspective. This is particularly useful on devices where screen real estate is limited. This functionality is only available on native Android and iOS apps for now. 

1.  Sign in to your Kissflow account using a native mobile app.
2.  Open any form containing a child table.
3.  Click the **More options** button (![](/migration-assets/form-child-tables-in-the-pwa-and-native-mobile-app/1.png)) at the top of the child table > click **Enter fullscreen**.
    
4.  Click the **Exit fullscreen** button (![](/migration-assets/form-child-tables-in-the-pwa-and-native-mobile-app/2.png)) to rotate the screen back to portrait mode.

### Editing data in a child table

1.  Tap any cell in the child table to launch the editing modal.
    
2.  Make necessary changes to the field values. Navigate through rows using the previous and next buttons within the modal.
    
3.  Changes are auto-saved.  
     
    

### Rearranging Columns

You can customize the order of columns according to your preference. To rearrange columns in a child table, follow these steps:

1.  Sign in to your Kissflow account using a PWA or native mobile app.
    
2.  Open any form containing a child table.
3.  Click the **More options** button (![](/migration-assets/form-child-tables-in-the-pwa-and-native-mobile-app/1.png)) at the top of the child table > click **Reorder columns**.
    
4.  Press and hold the column header you wish to move.
    
5.  Drag it to the desired position in the table.
    
6.  Release to set the column order. Click **Save**.
    
7.  You can click **Reset** to rearrange the columns to their original position.
    

### Deleting rows in bulk

Select and delete multiple rows at once from the table view, enhancing the manageability of large child tables.

1.  Sign in to your Kissflow account using a PWA or native mobile app.
    
2.  Open any form containing a child table.
3.  In the tabular view, tap the **selection** icon (![](/migration-assets/form-child-tables-in-the-pwa-and-native-mobile-app/3.png)) on the left of each row you want to delete. You can also long-press any row to trigger the bulk deletion action.
    
4.  Once all desired rows are selected, press the **Delete** button (![](/migration-assets/form-child-tables-in-the-pwa-and-native-mobile-app/4.png)) at the top of the table.
    
5.  Confirm deletion to remove the rows.
