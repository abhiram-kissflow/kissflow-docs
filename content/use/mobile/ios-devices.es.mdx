# Dispositivos iOS

En un dispositivo iOS, puedes acceder a tu cuenta de Kissflow mediante un navegador móvil o instalándola como una aplicación web progresiva (PWA).

## Usar un navegador móvil

Abre tu navegador móvil preferido y ve a la URL de tu cuenta: `<accountname>.kissflow.com`.

## Usar una aplicación web progresiva (PWA)

Una aplicación web progresiva te ofrece una experiencia similar a la de una aplicación nativa: se ve y funciona como una aplicación móvil, consume menos datos que la experiencia del navegador y se actualiza en segundo plano sin que tengas que hacer nada.

<Callout type="info">
  Para disfrutar de la mejor experiencia, usa un dispositivo con al menos 2 GB de RAM.
</Callout>

## Instalar la PWA en iOS

1. Abre **Safari**.
2. Escribe la URL de tu cuenta en la barra de direcciones: `https://<accountname>.kissflow.com`.
3. Toca el icono de **Acciones** (compartir).
4. Toca **Añadir a la pantalla de inicio**.
5. Toca **Añadir**.

<Callout type="warn">
  iOS impide que Chrome añada una PWA a la pantalla de inicio. Usa Safari para instalar la PWA.
</Callout>
