A [smart attachment](https://community.kissflow.com/t/y4h4az1/image-attachment-smart-attachment-and-signature-fields#smart-attachment-field) is an AI-powered field type in our forms that automatically extracts data from uploaded documents like receipts or invoices. It uses intelligent document processing to map the extracted information to corresponding fields in the form, saving users time on manual data entry.
