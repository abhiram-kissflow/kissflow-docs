**[Formula builder](/t/35h4alz/computing-fields-using-formula-builder)** is a built-in tool in Kissflow forms that helps you create and manage expressions more efficiently. It offers an intuitive editor with smart auto-suggestions when using the dot operator (.) or calling functions, and a user-friendly drag-and-drop interface for adding fields and functions.

The builder also includes a search function to locate specific fields or functions quickly, and lets you evaluate expressions before saving them. The formula builder simplifies the entire process of writing and editing expressions.

For Enterprise plans, an [AI formula builder](/t/83ylatd/ai-formula-builder) is available as an add-on.
