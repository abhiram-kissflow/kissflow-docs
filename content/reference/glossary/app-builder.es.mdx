**[Kissflow App Builder](/t/m1yzj9p/app-builder-overview)** es un espacio de trabajo dentro del entorno de desarrollo donde puedes ensamblar las páginas y los componentes de tu aplicación para crear una aplicación completa y funcional. Solo está disponible desde el entorno de desarrollo. Es decir, cuando haces clic en **Editar aplicación** en una aplicación publicada o no publicada, el constructor de aplicaciones se abre automáticamente.

El constructor de aplicaciones facilita el desarrollo simultáneo de varias funcionalidades, ya que puedes trabajar en una página mientras modificas un proceso en una pestaña independiente, a la que puedes acceder desde el panel izquierdo. Puedes cambiar de pestaña sin perder ningún progreso. También proporciona varios componentes predeterminados que puedes arrastrar y soltar en el lienzo, junto con diversas opciones de diseño y estilo para personalizar la apariencia de tu aplicación.
