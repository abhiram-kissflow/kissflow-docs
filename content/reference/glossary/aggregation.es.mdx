El [campo de agregación](https://community.kissflow.com/t/83h4azk/aggregation-field) reúne datos de varios campos para mostrar relaciones y realizar cálculos. Puedes usarlo para calcular sumas, promedios u otras agregaciones de datos dentro de un mismo formulario o entre distintos procesos. Puedes aplicar filtros para restringir los valores utilizados en la agregación, lo que te proporciona un mayor control sobre los resultados.
