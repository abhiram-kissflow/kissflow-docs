In the [analytics](/t/35h9qzj/analytics-reports-an-overview) module, child tables are automatically created for process and board. They are named in the ProcessName-ChildTableName or BoardName-ChildTableName format. For example, the Purchase Request process generates a child table named Purchase Request Catalog Items.
