[Round robin](/t/p8h9qt8/step-assignment-preference#round-robin) is an assignment method that you can choose under assignment preference for distributing tasks in a process. It assigns tasks to users in a rotating sequence. For example, if you have three users, the first task goes to User A, the second to User B, the third to User C, then back to User A, and so on. This ensures an even distribution of work.
