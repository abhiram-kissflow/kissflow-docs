En Analytics, las [tablas del sistema](/t/h7yq95d/system-table-in-analytics) se generan automáticamente para supervisar el estado y las actividades en fuentes de datos como procesos, tableros y conjuntos de datos. Algunos ejemplos son las tablas Activity Instance, Process Instance, Board Workflow, User y Group.
