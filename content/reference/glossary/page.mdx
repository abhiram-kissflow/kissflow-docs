Pages help you create interactive interfaces within the app builder that allow you to collect, display, and manage data visually. You can create multiple pages, tailor them to specific [user roles](/t/83h9q19/application-roles), and control access through permissions. Pages enable you to aggregate and present data in an organized, aesthetic format, offering a comprehensive overview on a single screen.
