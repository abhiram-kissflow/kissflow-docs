Un [dataview](/t/83h9qzn/understanding-dataviews) es una representación tabular de datos creada a partir de una o varias tablas u otros dataviews en Kissflow Analytics. Se forma realizando combinaciones o uniones mediante consultas SQL de Snowflake en el editor de consultas. Puedes crear informes basados en el dataview una vez que se haya generado.
