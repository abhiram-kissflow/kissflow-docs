Una aplicación es una solución de software que combina diversos elementos, como formularios, flujos de trabajo e integraciones, para llevar a cabo una o más tareas de un caso de uso complejo. Los usuarios pueden crear aplicaciones empresariales ensamblando visualmente estos componentes, a menudo con poca o ninguna programación. Puedes crear aplicaciones completamente funcionales y complejas con [Kissflow Apps](/t/x2h9qkj/introduction-to-kissflow-apps).
