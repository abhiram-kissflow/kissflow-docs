En Analytics, una [fuente de datos](/t/x2h9qzw/understanding-data-sources) hace referencia a cualquier forma de datos a partir de la cual puedes crear vistas de datos e informes. Puede ser una tabla de flujo, una child table o una tabla del sistema.
