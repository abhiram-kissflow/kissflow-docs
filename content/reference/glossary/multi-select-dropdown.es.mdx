Un campo de [menú desplegable de selección múltiple](https://community.kissflow.com/t/35h4azp/dropdown-multi-select-dropdown-checkbox-checklists-and-radio-buttons#multi-select-dropdown) es un elemento de formulario que permite a los usuarios elegir varias opciones de una lista, un conjunto de datos o un dataform de valores. Es similar a un menú desplegable normal, pero ofrece la funcionalidad adicional de seleccionar más de un elemento. Este tipo de campo es especialmente útil cuando necesitas recopilar varias selecciones de un conjunto predefinido de opciones.
