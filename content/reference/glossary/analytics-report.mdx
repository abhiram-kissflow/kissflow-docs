[Analytics reports](/t/m1h9qzd/overview-of-kissflow-analytics) allow you to create cross-form reports from various data sources, including process tables, child tables, system tables, datasets, and data views. These reports can be visualized in tabular or graphical formats. There are four types: Tabular reports, charts, card, and pivot reports.
