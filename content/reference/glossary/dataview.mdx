A [dataview](/t/83h9qzn/understanding-dataviews) is a tabular representation of data created from one or more tables or other dataviews in Kissflow Analytics. It is formed by performing joins or unions via Snowflake SQL queries in the query editor. You can create reports based on the dataview once it's generated.
