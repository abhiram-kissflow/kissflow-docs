[Lookup layout](/t/g9h4azr/lookup-field#configuring-lookup-layout) is a configuration setting that determines how lookup results are visually presented to the user within a [lookup field](/t/g9h4azr/lookup-field) in Forms. It controls the display format both during the selection and after an item has been chosen.
