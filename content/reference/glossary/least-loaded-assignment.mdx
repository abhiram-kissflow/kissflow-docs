[Least loaded](/t/p8h9qt8/step-assignment-preference#least-loaded) is an assignment method that you can choose under assignment preference for distributing tasks in a process. It assigns the task to the user with the fewest current assignments. This method aims to balance the workload among team members in real-time, preventing any one person from becoming overwhelmed.
