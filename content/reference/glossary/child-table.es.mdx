En el módulo de [analítica](/t/35h9qzj/analytics-reports-an-overview), las child table se crean automáticamente para procesos y tableros. Se les asignan nombres con el formato ProcessName-ChildTableName o BoardName-ChildTableName. Por ejemplo, el proceso Purchase Request genera una child table llamada Purchase Request Catalog Items.
