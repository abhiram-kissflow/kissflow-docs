# Administrar tableros en la PWA

Administra tus [tableros](/docs/build/boards/boards-overview) desde cualquier lugar usando Kissflow Progressive Web App (PWA). El panel de **Inicio** tiene dos widgets para trabajar con elementos de tableros:

- **Mis tareas** — muestra los elementos que tienes asignados.
- **Mis elementos** — muestra los elementos que creaste.

![Panel de Inicio con los widgets Mis tareas y Mis elementos](/migration-assets/managing-boards-in-pwa/1.png)
{/* TODO: verify screenshot matches current product UI */}

## Crear un elemento

1. En el panel, toca el botón **Nuevo elemento** (![Icono de Nuevo elemento](/migration-assets/managing-boards-in-pwa/2.png)) junto a un tablero.
2. En el formulario que se abre, completa los detalles — responsable, fecha de vencimiento, estado, prioridad y cualquier otro campo que registre tu tablero.
3. Toca **Enviar**.

<Callout type="info">
  También puedes acceder al formulario para crear un elemento en un tablero tocando **Explorar**, abriendo el tablero y tocando el botón **Nuevo elemento**.
</Callout>

![Crear un elemento nuevo desde el panel](/migration-assets/managing-boards-in-pwa/5.gif)
{/* TODO: verify screenshot matches current product UI */}

## Explorar elementos en un tablero

1. Toca **Explorar**.
2. Toca un tablero para ver sus elementos.
3. Cambia entre las pestañas para filtrar la vista:
   - **Todos** — todos los elementos del tablero. Esta es la pestaña predeterminada.
   - **Asignados a mí** — elementos que tienes asignados para realizar una acción.

<Callout type="info">
  Si eres [iniciador](/docs/build/boards/managing-board-roles-and-permissions) en el tablero, también verás una pestaña **Iniciados por mí** junto a **Todos**, fijada de forma predeterminada.
</Callout>

4. Toca el botón **Ordenar** (![Icono de Ordenar](/migration-assets/managing-boards-in-pwa/3.png)) para ordenar los elementos que se muestran en una pestaña.

![Ordenar elementos en una pestaña del tablero](/migration-assets/managing-boards-in-pwa/6.gif)
{/* TODO: verify screenshot matches current product UI */}

## Administrar un elemento

1. Abre un elemento del tablero.
2. Toca el botón **Más opciones** (![Icono de Más opciones](/migration-assets/managing-boards-in-pwa/4.png)).
3. Elige una acción:
   - **Copiar enlace** — copia una URL directa del elemento para compartirla con otras personas.
   - **Seguir** — [sigue](/docs/use/items/collaborating) un elemento que no tienes asignado para hacer un seguimiento de su progreso.
   - **Clonar** — crea un duplicado del elemento.
   - **Eliminar** — elimina el elemento.

![Menú Más opciones de un elemento que muestra Copiar enlace, Seguir, Clonar y Eliminar](/migration-assets/managing-boards-in-pwa/8.gif)
{/* TODO: verify screenshot matches current product UI */}

## Hacer un seguimiento de los elementos desde Mis tareas y Mis elementos

1. En el panel, toca el widget **Mis tareas** para ver los elementos que tienes asignados, o el widget **Mis elementos** para revisar el progreso de los elementos que creaste.
2. Toca un elemento para abrirlo. Los cambios que hagas en sus campos se guardan automáticamente.

También puedes agregar [notas](/docs/use/boards/notes-status-logs-and-activity-logs-in-boards) para colaborar con otras personas en el elemento. Las notas admiten @menciones, emojis, archivos adjuntos y otras opciones de formato.

![Abrir un elemento desde Mis tareas y agregar una nota](/migration-assets/managing-boards-in-pwa/7.gif)
