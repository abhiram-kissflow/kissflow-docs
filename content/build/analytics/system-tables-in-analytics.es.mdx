Planes de Kissflow:

✓ Basic ✓ EnterpriseLa tabla del sistema es una tabla que contiene detalles específicos de toda la información de carga de trabajo en Kissflow. Hay dos [tablas del sistema](/t/83yq9jv/process-table#system-tables) para los procesos llamadas ProcessInstance y ActivityInstance, que contienen el estado actual y el historial de actividades del flujo de trabajo de todos los elementos de todos los procesos. Los tableros también tienen una tabla del sistema llamada [Board Workload Instance](/t/83yqk6f/board-tables-in-analytics#board-workflow-instance-table), que contiene la información de carga de trabajo de todos los tableros.  
  
El siguiente diagrama sobre la relación entre las entidades de las fuentes de datos ayuda a comprender la estructura de las tablas de Analytics.![](/migration-assets/system-tables-in-analytics/1.png)Del mismo modo, hay dos tablas del sistema dedicadas exclusivamente a la información relacionada con los usuarios y grupos de tu cuenta de Kissflow. Estas tablas te proporcionan una vista detallada de las acciones realizadas por los usuarios añadidos a tu cuenta.

Puedes encontrar la tabla del sistema dentro de la pestaña Explorador de datos. También puedes ordenar las fuentes de datos según las tablas del sistema para identificarlas fácilmente en Analytics. Los campos de las tablas del sistema se pueden usar para generar varios tipos de informes, como informes tabulares, de gráficos y dinámicos.
