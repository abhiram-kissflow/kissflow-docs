## Fuentes de datos

Una fuente de datos es una colección de todos los datos presentes en los procesos, tableros, conjuntos de datos y otras tablas de Kissflow. Se utilizan para crear [informes](/t/35h9qzj/analytics-reports-an-overview) en Analytics. Aunque no puedes crear, modificar ni eliminar fuentes de datos, puedes combinarlas para [crear vistas de datos](/t/83h9qzn/understanding-dataviews#creating-a-dataview).

## Explorador de datos

Como [superadministrador](/t/35h4ak0/user-roles-permissions#super-admins), puedes ver todas las fuentes de datos de tu cuenta de forma predeterminada. Estas incluyen tablas de procesos, tablas de tableros, Child tables, conjuntos de datos, tablas del sistema y vistas de datos. Puedes buscar fácilmente una fuente de datos por su nombre, filtrar por tipo de fuente de datos y ordenar por última sincronización, última modificación o en orden alfabético.

Cada fuente de datos se muestra como una tarjeta con estos detalles:

-   Nombre.
-   Tipo.
-   Tiempo transcurrido desde la última sincronización.
-   Opción para crear un informe y compartir la fuente de datos con otras personas.

### Filtrado de fuentes de datos

Encuentra las fuentes de datos relevantes mediante los siguientes filtros:

-   **Todas**: muestra todas las fuentes de datos de tu cuenta.
-   **Mis vistas de datos**: muestra todas las vistas de datos que has creado.
-   **Vistas de datos compartidas**: muestra todas las vistas de datos que se han compartido contigo.
-   **Tablas del sistema**: muestra todas las tablas de carga de trabajo creadas por el sistema, incluidos los datos de usuarios y grupos.
-   **Mis tablas**: muestra todas las tablas de las que eres administrador.
-   **Tablas compartidas**: muestra todas las tablas a las que tienes acceso.

> **Nota:**
> 
> Si tienes el rol de administrador de procesos, administrador de tableros o administrador de informes para tu proceso o tablero, o si eres administrador o miembro de un conjunto de datos, adquirirás automáticamente estos permisos y te convertirás en administrador de informes de tu fuente de datos.

> **Nota:**
> 
> Si eres desarrollador, miembro de un proceso o tablero con permiso para crear informes, administrador de datos de un proceso o tablero específico, o visor del conjunto de datos, heredarás estos permisos y se te designará automáticamente como creador de informes de la fuente de datos.
