## ¿Qué es el conector de Kissflow para BI?

**El conector de Kissflow para BI** te permite optimizar tus operaciones empresariales existentes mediante la medición y el análisis de cualquier dato cualitativo y cuantitativo de Kissflow guardado en el almacén de datos en la nube de Snowflake, utilizando herramientas de inteligencia empresarial como Microsoft Power BI, Tableau, etc.

### Primeros pasos

El conector de Kissflow para BI se configura en [Snowflake](https://www.snowflake.com/en/data-cloud/workloads/data-warehouse/), una plataforma de almacenamiento de datos donde se guardan los datos de tu cuenta de Kissflow para fines analíticos. No podrás realizar ningún cambio en los datos de Snowflake porque son de solo lectura. Al utilizar Snowflake como conector, puedes acceder a estos datos y analizarlos con la herramienta de inteligencia empresarial (BI) que prefieras, como [Microsoft Power BI](https://powerbi.microsoft.com/en-au/) o [Tableau](https://www.tableau.com/).

Comencemos a configurar Snowflake y a conectarlo con herramientas de BI para crear paneles interactivos basados en tus datos de Kissflow.
