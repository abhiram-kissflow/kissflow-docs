## Fuentes de datos

Una fuente de datos es una colección de todos los datos presentes en los procesos, tableros, conjuntos de datos y otras tablas de Kissflow. Se utilizan para crear [informes](/t/35h9qzj/analytics-reports-an-overview) en Analytics.

Para acceder a las fuentes de datos:

1.  Inicia sesión en Kissflow.
2.  Haz clic en el botón **Analytics** (![](/migration-assets/overview-of-kissflow-analytics/2.png)) en la navegación izquierda.
3.  En **Explorador de datos**, puedes encontrar todas las **fuentes de datos**, incluidas las tablas de procesos, tablas de tableros, child tables, tablas del sistema, vistas de datos y conjuntos de datos creados por ti o compartidos contigo.

![](/migration-assets/understanding-data-sources-2/1.png)

> **NOTA:**
> 
> -   Aunque no puedes crear, modificar ni eliminar fuentes de datos, puedes combinarlas para crear [vistas de datos](/t/83h9qzn/understanding-dataviews#creating-a-dataview).
> -   Todas las fuentes de datos se sincronizarán en un plazo de una hora. La marca **Última sincronización** indica la hora de la sincronización más reciente.

Como [Superadministrador](/t/35h4ak0/user-roles-permissions#super-admins), puedes ver todas las fuentes de datos de tu cuenta de forma predeterminada. Estas incluyen tablas de procesos, tablas de tableros, child tables, conjuntos de datos, tablas del sistema y vistas de datos. Puedes buscar fácilmente una fuente de datos por su nombre, filtrar por tipo de fuente de datos y ordenar por última sincronización, última modificación o por orden alfabético.

Cada fuente de datos se muestra como una tarjeta con estos detalles:

-   Nombre.
-   Tipo.
-   Tiempo transcurrido desde la última sincronización.
-   Opción para crear un informe y compartir la fuente de datos con otras personas.

![](/migration-assets/understanding-data-sources-2/2.png)

### Acceder a una fuente de datos por primera vez

Cuando se accede a una fuente de datos por primera vez, se realiza una obtención de datos única, como se muestra en el gif a continuación. Posteriormente, la fuente de datos se programa para sincronizarse periódicamente y mantener los datos actualizados. Esta obtención de datos inicial normalmente se completa en un plazo de cero a diez minutos, según el tamaño y la complejidad de la fuente de datos.

## Filtrar en el explorador de datos

Encuentra fuentes de datos relevantes mediante los siguientes filtros:

-   **Todas** - Muestra todas las fuentes de datos de tu cuenta.
-   **Mis vistas de datos** - Muestra todas las vistas de datos que has creado.
-   **Vistas de datos compartidas** - Muestra todas las vistas de datos que se han compartido contigo.
-   **Tablas del sistema** - Muestra todas las tablas de carga de trabajo creadas por el sistema, incluidos los datos de usuarios y grupos.
-   **Mis tablas** - Muestra todas las tablas de las que eres administrador.
-   **Tablas compartidas** - Muestra todas las tablas a las que tienes acceso.

> **Nota:**
> 
> Si tienes el rol de administrador de procesos, administrador de tableros o administrador de informes para tu proceso o tablero, o si eres administrador o miembro de un conjunto de datos, adquirirás automáticamente estos permisos y te convertirás en administrador de informes de tu fuente de datos.

> **Nota:**
> 
> Si eres desarrollador o miembro de un proceso o tablero con permiso para crear informes, administrador de datos de un proceso o tablero específico, o visor del conjunto de datos, heredarás estos permisos y se te designará automáticamente como creador de informes de la fuente de datos.
