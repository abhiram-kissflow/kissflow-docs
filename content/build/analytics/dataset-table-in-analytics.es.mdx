Las tablas de conjuntos de datos aparecen en Kissflow Analytics para cada conjunto de datos que crees en tu cuenta de Kissflow. Estas tablas están diseñadas con una [estructura](/media/download/632kc6/KF_analytics_schema.pdf) tabular, con filas y columnas que almacenan valores de datos de manera organizada.  
  
El diagrama que muestra la relación entre las entidades de las fuentes de datos te ayuda a comprender Analytics.

![](/migration-assets/dataset-table-in-analytics/1.png)

JSON

Un subcampo de "\_created\_by" siempre contiene el valor "User", lo que indica que un usuario creó el elemento.

Otro subcampo de "\_created\_by" muestra el nombre del usuario que creó el elemento en un formato que incluye tanto su nombre como su apellido.

\_created\_at.display\_value

Creado el

DateTime

Representa la fecha y hora de creación del elemento en formato GMT en la tabla del conjunto de datos.

\_created\_at.timedelta

Creado el

DateTime

Este campo contiene la duración de la diferencia de zona horaria entre la hora media de Greenwich (GMT) y la zona horaria específica. Permanece vacío cuando no existe ninguna diferencia.

\_created\_at.timezone

Creado el

DateTime

\_created\_at.value

Creado el
