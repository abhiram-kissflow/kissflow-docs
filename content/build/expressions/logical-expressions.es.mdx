Las expresiones lógicas son booleanas, lo que significa que devuelven un valor TRUE o FALSE. 

Los valores booleanos no se muestran directamente en un formulario. Se utilizan en dos situaciones: 

-   La prueba lógica dentro de una función IF.
-   Para establecer la condición que determina si un paso o una rama del flujo de trabajo se ejecuta o no.

## Funciones

**Función**

**Descripción**

**Sintaxis**

**Valor**

and

TRUE si ambas expresiones booleanas son verdaderas.

FALSE si alguna de las expresiones es falsa.

Solo se permiten dos parámetros.

AND(num1 = 6, num2 = 4)

True

&&

Igual que la función AND, pero permite usar varias expresiones.

num1 = 6 && num2 = 4 && text1 = "Tony" 

False

or

TRUE si alguna de las dos expresiones booleanas es verdadera.

FALSE si ninguna de las expresiones es verdadera.

Solo se permiten dos parámetros.

OR(num1 = 6, num2 = 10)

True

\\|\\|

Igual que la función OR, pero permite usar varias expresiones.

num1 = 6 \\|\\| num2 = 10 \\|\\| text1 = "Tony"

True

isBlank

Comprueba si un campo está vacío (TRUE) o tiene un valor (FALSE).

Se puede usar con cualquier tipo de campo.

isBlank(text1)
