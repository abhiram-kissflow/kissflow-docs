When developing a business app, you would prefer tools that aid in quick, tangible visual techniques. The page creation tool in the app builder uses a simple drag-and-drop interface to create powerful visual interface. You can also add JavaScript code elements and events to evoke special features in your pages.

## Components available in the page builder

You can explore multiple pre-defined set of components in each of these categories and drop them onto the page grid. You can add color, insert background images, set styles and define the layout for each of these components.

## Adding views and reports

In addition to components, you can add forms or form views such as table, gallery, and sheet to your page.

You can also add table, chart, pivot, and card reports as well as performance metrics reports.

To add a form, view, or a report to a page, drag and drop the component to the page.

In the **Settings** tab on the right, use the dropdown menus to select a flow type, a specific flow, and a specific report or a view that you have already created. If the selected type of flow (dataform, board, process) does not exist yet, click the **Create new** option to create a new one.
