#### Introduction

A hyperlink component lets you add links to your page. On clicking the hyperlink, the users can access additional information related to the given content.

![](/migration-assets/hyperlink/1.gif)

#### How to use it?

Once you drag and drop the hyperlink component, you will be able to perform the following actions:

Under **General Properties**, you will be able to

-   Edit the caption.
-   Enter the URL to which you want the user to be redirected to.
-   Choose if the URL should open in a new tab using the toggle button.
-   Set the font size, color, background color, text style such as Bold, Italics, Underline and Strikethrough, and tooltip.

In [Style properties](/t/y4yzj9m/managing-component-style-properties), you will be able to adjust the sizing, modify the padding and margin, and set the border size, color, and radius.
