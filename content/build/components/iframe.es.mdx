#### Introducción

Un marco en línea, comúnmente denominado iframe, es un elemento HTML que se utiliza para insertar otra página web dentro de la página actual. El elemento iframe obtiene el código de la fuente referenciada y lo renderiza dentro de esa página.

![](/migration-assets/iframe/1.png)

#### ¿Cómo se usa?

El componente iframe se arrastra y se suelta en el generador de páginas. La URL de origen consta de una URL existente insertada en la página. Puedes modificarla para insertar la URL de origen de la página web que deseas insertar.

En [Propiedades de estilo](/t/y4yzj9m/managing-component-style-properties), podrás ajustar el tamaño, modificar el relleno y el margen, y establecer el tamaño, el color y el radio del borde.
