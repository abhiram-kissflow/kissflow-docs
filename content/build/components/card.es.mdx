#### Introducción

Una tarjeta se puede configurar para mostrar datos importantes, como los activos disponibles, el costo de los activos disponibles o las solicitudes pendientes con el personal administrativo. Se puede usar para realizar un seguimiento de las tareas o para otros casos de uso, como mostrar los detalles de un dominio específico.

![](/migration-assets/card/1.gif)

#### ¿Cómo usarla?

Una vez que arrastres y sueltes el componente de tarjeta, podrás realizar las siguientes acciones:

-   Introduce cualquier texto, como Tareas asignadas, Tareas vencidas, etc., en el campo de texto.
-   Elige un estilo de tarjeta entre los diferentes estilos disponibles.
-   Puedes introducir el número de tareas y configurar el icono de la tarjeta.
-   Modifica el color, el color del texto, el color de fondo y el fondo del icono.
-   En [Propiedades de estilo,](/t/y4yzj9m/managing-component-style-properties) podrás alinear la tarjeta, ajustar el tamaño, modificar el relleno y el margen, y establecer el tamaño, el color y el radio del borde.
-   En Propiedades de eventos, podrás añadir el evento “**Al hacer clic**” para que la tarjeta realice una acción.

**Se pueden realizar tres acciones:**

1. **Redirección**: esta acción te redirige a las páginas definidas por el usuario o a la página de inicio definida por el sistema, o te permite volver a la misma página.
2. **Abrir ventana emergente**: la acción de abrir una ventana emergente te permite abrir una ventana emergente que hayas creado usando el componente [Ventana emergente](/t/35yzj4w/popup).
3. **Acción de JavaScript**: la acción de JavaScript te permite definir la acción que quieres realizar usando código JS al hacer clic en la tarjeta.
