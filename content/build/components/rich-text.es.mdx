## ¿Qué es el texto enriquecido?

El componente de **Texto enriquecido** te permite agregar contenido de texto con estilos directamente a tu página. Admite títulos, párrafos, enlaces, listas, imágenes y formato básico (como negrita, cursiva y subrayado), por lo que es ideal para agregar contenido informativo o descriptivo sin escribir código.

Puedes usar el componente de texto enriquecido para insertar instrucciones, descripciones, etiquetas o cualquier otro contenido estático que mejore el diseño de tu página. Es totalmente personalizable y puedes aplicarle estilos para que coincida con la apariencia de tu aplicación.

## ¿Cómo se usa?

1.  Arrastra y suelta el componente de texto enriquecido en tu página.
    
2.  Haz clic dentro del componente para escribir el texto directamente en el editor de texto integrado.
    
3.  Escribe una barra diagonal / para abrir el menú desplegable de comandos. Aquí encontrarás opciones de formato y estilo.
    
4.  Usa la opción **Campos** del menú desplegable de comandos para insertar variables y parámetros.
    

### Uso de comandos para texto enriquecido

El menú desplegable de comandos incluye las siguientes opciones:

-   **Formato en línea**: negrita, cursiva, subrayado, tachado, subíndice y superíndice.
    
-   **Estructura del texto**: títulos (H1, H2), párrafo y cita en bloque.
    
-   **Listas**: listas con viñetas y numeradas.
    
-   **Enlaces y contenido multimedia**: agrega hipervínculos, imágenes o contenido insertado.
    
-   **Fragmento de código**: te proporciona un bloque de código para agregar un fragmento de código a tu página.
    
-   **Compatibilidad con variables**: la opción **Campos** te ayuda a insertar valores dinámicos mediante variables locales o globales.
    

![](/migration-assets/rich-text/2.gif)

> **Nota:** Por motivos de seguridad, el componente de texto enriquecido trata los fragmentos de código pegados en él como texto sin formato y no los interpreta como código.

### Visibilidad

En la pestaña **Visibilidad**, puedes establecer una visibilidad predeterminada y agregar reglas de visibilidad para el componente de texto enriquecido si deseas mostrarlo u ocultarlo en determinadas condiciones.

### Propiedades de estilo

En [Propiedades de estilo,](/t/y4yzj9m/managing-component-style-properties) puedes elegir entre diversas opciones de formato para el componente de texto enriquecido. Podrás ajustar el tamaño, modificar el relleno y el margen, y establecer el tamaño, el color y el radio del borde; estos cambios se aplicarán al componente en su totalidad.

Selecciona el texto dentro del editor para abrir la barra de herramientas de formato y aplicar formato exclusivamente al texto del editor de texto enriquecido.
