#### Introduction

An inline frame, commonly referred to as the iframe, is an HTML element used to embed another webpage within the current page. The iframe element gets the code from the referenced source and is rendered within that page.

![](/migration-assets/iframe/1.png)

#### How to use it?

The iframe component is dragged and dropped into the page builder. The source URL consists of a pre-existing URL embedded within the page. It can be modified to embed the source URL of the webpage that you wish to embed.

In [Style properties](/t/y4yzj9m/managing-component-style-properties), you will be able to adjust the sizing, modify the padding and margin, and set the border size, color, and radius.
