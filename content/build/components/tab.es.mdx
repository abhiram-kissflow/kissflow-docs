#### Introducción

Un componente de pestañas te permite categorizar el contenido en diferentes pestañas. Con un contenedor de pestañas, puedes crear una barra de navegación horizontal con pestañas individuales que tienen su propio contenedor (enlace). Básicamente, te permite crear una navegación secundaria para tu aplicación.

![](/migration-assets/tab/1.gif)

#### ¿Cómo se usa?

Una vez que arrastres y sueltes el componente de pestañas, podrás realizar las siguientes acciones:

1.  Elige un estilo de pestaña.
2.  En la configuración de las pestañas, puedes:

-   Editar el nombre de la pestaña.
-   Establecer una pestaña determinada como la pestaña predeterminada.
-   Añadir pestañas adicionales.

1.  Establece el color, el color activo y el color al pasar el cursor por encima de la pestaña.
2.  En las propiedades de estilo, podrás ajustar el tamaño, modificar el relleno y el margen, y establecer el tamaño, el color y el radio del borde.
3.  En las propiedades de eventos, podrás añadir el evento “Al cambiar de pestaña” para indicar que la pestaña realice una acción.

**Hay tres acciones que se pueden realizar:**

1. **Redirección**: esta acción te redirige a la página de inicio definida por el usuario, a la página de inicio definida por el sistema o te permite volver a la misma página.
2. **Abrir ventana emergente**: la acción de abrir una ventana emergente te permite abrir una ventana emergente que hayas creado usando el componente [Ventana emergente](/t/35yzj4w/popup).
3. **Acción de JavaScript**: la acción de JavaScript te permite definir la acción que quieres realizar mediante código JS.
