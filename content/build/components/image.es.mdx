#### Introducción

Al utilizar un componente de imagen, podrás añadir imágenes a tu aplicación.

![](/migration-assets/image/1.png)

#### ¿Cómo usarlo?

Las imágenes pueden añadirse a la página mediante URLs.

Puedes añadir la URL de la imagen en el origen de la imagen y elegir el estilo de imagen entre las siguientes opciones:

1. **Contain**: En el modo contain, la imagen cambia de tamaño para ajustarse a las dimensiones indicadas, pero conserva su relación de aspecto.
2. **Cover**: En el modo cover, la imagen conserva su relación de aspecto y ocupa las dimensiones indicadas.
3. **Fill**: Al seleccionar fill, la imagen cambia de tamaño para ocupar las dimensiones indicadas. 
4. **None**: En este caso, la imagen no cambia de tamaño.  

En las [propiedades de estilo](/t/y4yzj9m/managing-component-style-properties), podrás ajustar el tamaño, modificar el relleno y el margen, y establecer el tamaño, el color y el radio del borde.

En las **propiedades de eventos**, podrás añadir el evento “**Al hacer clic**” para que la imagen realice una acción. 

**Se pueden realizar tres acciones:**

1. **Redirección** - Esta acción te redirige a las páginas definidas por el usuario, a la página de inicio definida por el sistema o te permite volver a la misma página.
2. **Abrir ventana emergente** - La acción de abrir una ventana emergente te permite abrir una ventana emergente que hayas creado mediante el componente [Ventana emergente](/t/35yzj4w/popup).
3. **Acción de JavaScript** - La acción de JavaScript te permite definir la acción que quieres realizar mediante código JS.
