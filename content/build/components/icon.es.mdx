#### Introducción

Los iconos son elementos interactivos que se utilizan en las aplicaciones con fines de interfaz de usuario. Un componente de icono te permite añadir iconos a tu página.

#### ¿Cómo se usa?

-   Después de arrastrar y soltar el componente de icono, aparecerá un icono estático. Puedes modificarlo proporcionando una URL de icono diferente para establecer el icono que prefieras.![](/migration-assets/icon/1.gif) 
-   Puedes elegir un color de fondo para el icono desde la paleta de colores.
-   Para añadir una insignia al icono, activa el botón de alternancia de insignia. Las insignias se pueden establecer como un contador o como un punto.
-   Si estableces una insignia como contador, puedes definir manualmente el valor del contador o llamarlo mediante variables globales/locales y parámetros de entrada.
-   Puedes establecer el color de la insignia desde la paleta de colores.
-   También puedes añadir una descripción emergente para el icono y posicionarla.
-   En [Propiedades de estilo](/t/y4yzj9m/managing-component-style-properties), podrás ajustar el tamaño, modificar el relleno y el margen, y establecer el tamaño, el color y el radio del borde. [Obtén más información](/t/m1ykbax/working-with-page-layouts#setting-up-icon-sizing) sobre cómo funciona el tamaño de los iconos en el diseño de cuadrícula.
-   En **Propiedades de eventos**, podrás añadir el evento “**Al hacer clic**” para indicar que el icono realice una acción.

##### Hay tres acciones que se pueden realizar:

1. **Redirección**: esta acción te redirige a las páginas definidas por el usuario, a la página de inicio definida por el sistema o te permite volver a la misma página.
2. **Abrir ventana emergente**: la acción de abrir una ventana emergente te permite abrir una ventana emergente que hayas creado mediante el componente [Ventana emergente](/t/35yzj4w/popup).
3. **Acción de JavaScript**: la acción de JavaScript te permite definir la acción que quieres realizar mediante código JS.
