## ¿Qué es un divisor?

El componente **Divider** se utiliza para crear una separación visual entre las secciones de una página. Puedes usarlo en las páginas de tu aplicación para dividir el contenido en áreas claramente definidas.

Los divisores mejoran la jerarquía visual y facilitan el análisis y la comprensión de diseños complejos. Son especialmente útiles para agrupar elementos relacionados o separar secciones distintas, como encabezados, pies de página, barras laterales o diferentes bloques de contenido.

## ¿Cómo se utiliza?

Arrastra y suelta el componente **Divider** en tu página y colócalo entre contenedores o componentes. Puedes reorganizar los componentes como quieras mediante el árbol de componentes.

### Propiedades generales

**Orientación**: Elige entre **Horizontal** o **Vertical** en la pestaña **General**. El divisor se ajustará automáticamente según tu selección.  
**Etiqueta**: Si quieres, añade una etiqueta significativa para proporcionar contexto. Puedes escribir el texto directamente en el campo o insertar una variable global o local haciendo clic en el icono +.

### Visibilidad

En la pestaña **Visibilidad**, puedes establecer una visibilidad predeterminada y añadir reglas de visibilidad para el divisor si quieres mostrarlo u ocultarlo en determinadas condiciones.

![](/migration-assets/divider/1.png)

### Propiedades de estilo

En [Propiedades de estilo,](/t/y4yzj9m/managing-component-style-properties) puedes elegir entre varias opciones de formato tanto para el divisor como para su etiqueta. Podrás ajustar el tamaño, modificar el relleno y el margen, y establecer el tamaño, el color y el radio del borde. Para la etiqueta, establece el tamaño de fuente, el color y el color de fondo, y personaliza el estilo del texto.
