#### Introducción

Puedes usar una etiqueta para añadir un texto específico a tu página. Por ejemplo, puedes tener diferentes secciones en tu página y usar distintas etiquetas de texto para identificarlas.

![](/migration-assets/label/1.png)

#### ¿Cómo se usa?

Una vez que arrastres y sueltes el componente Label, podrás realizar las siguientes acciones:

1. Editar el texto de etiqueta preexistente y cambiarlo por el texto que quieras.
2. Establecer el tamaño y el color de la fuente, el color de fondo y el estilo del texto, como negrita, cursiva, subrayado y tachado.

En [las propiedades de estilo](/t/y4yzj9m/managing-component-style-properties) podrás ajustar el tamaño, modificar el relleno y el margen, y establecer el tamaño, el color y el radio del borde.
