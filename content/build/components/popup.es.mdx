#### Introducción

El componente de ventana emergente te permite añadir ventanas emergentes a tu página. Una ventana emergente es una pequeña ventana que aparece sobre la página existente al realizar una acción. Cada componente de ventana emergente contiene un encabezado y un contenedor.

![](/migration-assets/popup/1.png)

#### ¿Cómo usarlo?

Una vez que arrastres y sueltes el componente de ventana emergente, la ventana se abrirá y podrás realizar las siguientes acciones:

-   Modificar el nombre de la ventana emergente, el título del encabezado y el color del texto.
-   El tipo de fondo se puede establecer como un color o una imagen.
-   Se pueden añadir parámetros a la ventana emergente y establecer el tipo de parámetro como número o texto.
-   En [Propiedades de estilo](/t/y4yzj9m/managing-component-style-properties), podrás ajustar el tamaño, modificar el relleno y el margen, y establecer el tamaño, el color y el radio del borde.
-   En las propiedades de eventos, podrás añadir los eventos «**Al abrir**» y «**Al cerrar**» para indicar que la ventana emergente realice una acción. 

Se pueden realizar dos acciones:

1. **Redirección**: esta acción te redirige a las páginas definidas por el usuario, a la página de inicio definida por el sistema o te permite volver a la misma página.
2. **Acción de JavaScript**: la acción de JavaScript te permite definir la acción que quieres realizar mediante código JS.

Las ventanas emergentes se pueden vincular como un evento a otros componentes, como [botones](/t/x2yzj42/button), [tarjetas](/t/35yzj4v/card) y [pestañas](/t/p8yzj4s/tab).
