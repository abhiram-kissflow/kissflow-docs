#### Contenedor

Un contenedor es una herramienta que se utiliza para crear un conjunto de componentes dentro de un espacio definido. Dentro de un contenedor, podrás agregar diferentes componentes disponibles en el constructor de páginas. Puedes configurar dos tipos de diseños de página para el contenedor: [Flex y Grid](/t/m1ykbax/working-with-page-layouts).

![](/migration-assets/container/1.png)

#### ¿Cómo usarlo?

Los contenedores se pueden utilizar como el cuerpo de ese espacio definido. Te permiten agregar varios componentes, como [botones](/t/x2yzj42/button), [etiquetas](/t/g9yzj40/label), etc., dentro del contenedor.

Cada contenedor tiene un ID de contenedor único específico para ese contenedor. 

**Las siguientes acciones se pueden realizar en las** [**propiedades de estilo**](/t/y4yzj9m/managing-component-style-properties) **del contenedor:** 

1. Modificar la alineación.
2. Establecer el tamaño.
3. Ajustar el relleno y el margen.
4. Establecer el tamaño, el color y el radio del borde.
5. Aplicar sombra y color de fondo.

> **Nota:** No se pueden activar eventos en un contenedor.
