#### Introducción

Un botón es un componente clave de cualquier aplicación, ya que permite a los usuarios enviar sus preferencias. Puedes añadir botones para dirigirte a una página diferente o abrir una ventana emergente en función de eventos de clic específicos. También puedes usar código JavaScript (JS) en el componente de botón para ejecutar acciones específicas dentro de la página.

![](/migration-assets/button/1.png)

#### ¿Cómo usarlo?

A diferencia de otros componentes, un botón se utiliza únicamente cuando es necesario, ya que solicita al usuario que realice una acción. Una vez que arrastres y sueltes el componente de botón, podrás realizar las siguientes acciones:

-   Cambiar el texto del botón. Algunos textos de botón de uso común son Enviar, Compartir, Aceptar, Rechazar, Aprobar, Cancelar, etc.
-   Modificar el estilo del botón, incluido el tamaño, el color de fondo y el color del texto.
-   En **Propiedades de estilo**, podrás alinear el botón, ajustar el tamaño, modificar el relleno y el margen, y establecer el tamaño, el color y el radio del borde.
-   En **Propiedades de eventos**, podrás añadir el evento “**Al hacer clic**” para indicar al botón que realice una acción. 

##### Acciones que puedes realizar

1. **Redirección** - Esta acción te redirige a las páginas definidas por el usuario, a la página de inicio definida por el sistema o te permite volver a la misma página.
2. **Abrir ventana emergente**\- La acción de abrir una ventana emergente te permite abrir una ventana emergente que hayas creado con el componente [Ventana emergente](/t/35yzj4w/popup).
3. **Acción de JavaScript**\- La acción de JavaScript te permite definir la acción que quieres realizar mediante código JS al hacer clic en el botón.

Los botones también se pueden utilizar dentro de otros componentes, como [Contenedor](/t/q6yzj4n), [Detalles maestros](/t/83yzj44/master-details), [Pestaña](/t/p8yzj4s/tab) y [Ventana emergente](/t/35yzj4w/popup).
