# Safelisting IPs for external data connections

<PlanBadge plans={["enterprise"]} />

Some external platforms only accept connections from IP addresses you've explicitly trusted. Safelisting Kissflow's outbound IPs in those platforms tells them to treat Kissflow as a trusted source.

Kissflow sends outbound calls — including calls to external platforms from your integrations — through [Google Cloud NAT](https://cloud.google.com/nat/docs/overview), using a fixed set of reserved static IPs.

<Callout type="info">
  Not every platform requires safelisting. Check the external platform's own documentation first — if it doesn't require it, you can skip this page.
</Callout>

Find your account's domain below and safelist the matching IPs.
