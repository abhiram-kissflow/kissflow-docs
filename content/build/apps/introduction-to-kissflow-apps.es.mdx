## ¿Qué es una aplicación de Kissflow?

Kissflow Apps utiliza un enfoque de desarrollo de aplicaciones low-code que requiere poca o ninguna programación para crear aplicaciones. Utiliza herramientas visuales y scripts personalizados para desarrollar y ofrecer aplicaciones complejas a gran escala.

Puedes utilizar el framework de Kissflow Apps para crear aplicaciones y flujos de trabajo dinámicos para usuarios sobre Kissflow. Puedes usarlos para crear vistas de paneles personalizadas para Kissflow Processes y Boards, desarrollar integraciones sólidas o mostrar diseños de informes y dataforms en Kissflow.

Según los roles de usuario especificados, los usuarios pueden interactuar con estas aplicaciones dentro de su Kissflow Work Platform. Tus aplicaciones pueden ser completamente del lado del cliente o realizar solicitudes a recursos remotos.

> **Nota:** Ten en cuenta que el generador de aplicaciones solo está disponible en el plan **Kissflow Work Platform Enterprise**.

### ¿En qué se diferencia del desarrollo de aplicaciones tradicional?

En lugar de utilizar lenguajes de programación tradicionales como Java, C++, Python, etc., la plataforma de aplicaciones de Kissflow cuenta con funciones de arrastrar y soltar y componentes preconfigurados que permiten desarrollar aplicaciones con poca o ninguna programación. También puedes utilizar distintas variantes de JavaScript para añadir funciones personalizadas a tu aplicación. Esto no solo facilita el desarrollo de aplicaciones y lo hace más accesible para desarrolladores no profesionales, sino que también acelera significativamente todo el proceso de desarrollo.

Además de tu cuenta principal de Kissflow, tendrás acceso a entornos independientes de desarrollo y pruebas para crear y probar tus aplicaciones de forma segura.

### ¿Cuándo utilizar Kissflow Apps?

Las aplicaciones desempeñan un papel importante en la automatización de los flujos de trabajo. Processes en Kissflow te permiten implementar y ejecutar pequeños flujos de procesos en poco tiempo. Esto resulta muy práctico si necesitas comenzar rápidamente y tienes un grupo muy reducido de personas utilizando el flujo. Sin embargo, si tu caso de uso requiere una aplicación completa y tienes un gran número de partes interesadas, desarrollar una aplicación low-code es la mejor opción.

Por ejemplo, cada herramienta utilizada en los flujos de trabajo, como la gestión de procesos, la gestión de casos o la gestión de documentos, es una aplicación. Por lo general, están alojadas en una plataforma como un software BPM, CRM o ERP. Además, se conectan con software de terceros, como Google Sheet, Gmail, Slack, etc., mediante integraciones. Si tu empresa se encuentra en alguna de las categorías anteriores, deberías considerar utilizar el generador de aplicaciones de Kissflow para crear una aplicación.

Con Kissflow Apps, las empresas pueden crear aplicaciones personalizables y permitir que desarrolladores no profesionales creen aplicaciones en muy poco tiempo. Cualquier persona de una empresa, desde responsables de RR. HH. hasta personal administrativo y representantes de ventas, puede desarrollar nuevas aplicaciones en un periodo relativamente corto y sin necesidad de conocimientos profesionales de programación. Esto permite a la empresa acelerar la implementación de aplicaciones, impulsando la transformación digital y optimizando los procesos empresariales para promover la eficiencia y el crecimiento.
