Planes de Kissflow:

✕ Basic ✓ Enterprise

El [SDK de Kissflow](https://developers.kissflow.com/gettingstarted/) proporciona un conjunto sólido de métodos del SDK para ayudarte a crear aplicaciones low-code eficientes con facilidad. Este documento describe los diversos métodos del SDK disponibles para usar en formularios, aplicaciones e integraciones mediante JavaScript. Estos métodos te permiten recuperar información sobre las variables y los componentes de tu aplicación, lo que facilita una personalización fluida y una funcionalidad mejorada.

Este SDK te resultará útil si quieres:

1.  Diseñar e importar componentes de página personalizados usando HTML simple o frameworks de frontend como React, para hacerlos reutilizables dentro de tu aplicación.
2.  Ejecutar integraciones automatizadas que utilicen tus propios servicios externos y lógica empresarial para acceder a los datos de tu cuenta de Kissflow desde tu aplicación. El kit de desarrollo de software (SDK) de JavaScript para Kissflow Low-code incluye todas las funciones compatibles y los fragmentos de código que puedes incorporar a tu aplicación al integrarla con otros servicios.

Para habilitar la interacción entre tus componentes personalizados y tu aplicación de Kissflow, puedes instalar el SDK de Kissflow en tu proyecto. [Obtén más información](https://developers.kissflow.com/components/page/) sobre cómo instalar el paquete y crear componentes.

> Obtén más información sobre las distintas funciones del [SDK de JavaScript de las aplicaciones de Kissflow](https://developers.kissflow.com/gettingstarted/).
