El formato de los campos **Fecha** y **Fecha y hora** se basa en la configuración regional establecida por el usuario. Si no se ha establecido ninguna, se utilizará la configuración regional de la cuenta.

## Campo de fecha

El campo Fecha te permite seleccionar una fecha en el selector de fechas.

### Configuración

Puedes establecer una fecha predeterminada para que se muestre en el campo cuando el formulario se cargue por primera vez.

### Validaciones

Hay nueve tipos de validaciones que puedes usar para un campo de fecha. [Obtén más información sobre todas las validaciones](/t/m1h4az4/field-validations).

### Expresiones/fórmulas

Hay muchas [expresiones y fórmulas que puedes usar con un campo de fecha](/t/p8h4al3/date-expressions).

## Campo de fecha y hora

El campo Fecha y hora funciona exactamente como un campo de fecha y te permite recopilar la hora.

Al pasar el cursor sobre el campo, se mostrará la zona horaria. Mostrará la zona horaria establecida por el usuario o, si el usuario no ha establecido una zona horaria, la zona horaria de la cuenta.

### Configuración

Puedes establecer una fecha y hora predeterminadas para que se muestren en el campo cuando el formulario se cargue por primera vez.

### Validaciones

Puedes usar nueve tipos de validaciones para un campo de fecha y hora. Obtén más información sobre todas las [validaciones](/t/m1h4az4/field-validations).

### Expresiones/fórmulas

Hay muchas [expresiones y fórmulas que puedes usar con un campo de fecha y hora](/t/p8h4al3/date-expressions).
