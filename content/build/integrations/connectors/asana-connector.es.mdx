## ¿Qué es Asana?

Asana es una plataforma web y móvil de gestión del trabajo diseñada para ayudar a los equipos a organizar, supervisar y gestionar su trabajo. Para mantenerte al día con las prioridades del proyecto, puedes organizar tus tareas en una vista de tablero, lista o cronología.

## Conexión con Asana

El conector de Asana de Kissflow Integration te permite conectar y sincronizar tus proyectos y tareas de Asana con aplicaciones de terceros populares y aplicaciones de Kissflow. Debes tener una cuenta válida de Asana y ser cliente de pago de Kissflow para poder integrar Asana. Supón que ejecutas una campaña de marketing por correo electrónico a través de MailChimp y quieres realizar determinadas acciones en tus nuevos suscriptores de correo electrónico. Conectar Asana y MailChimp te permite crear fácilmente nuevas tareas para tus nuevos suscriptores de correo electrónico. Para realizar acciones específicas, usa el conector de Asana para recibir datos de otros flujos o aplicaciones. 

### Activadores del conector de Asana

**Eventos del activador**

**Descripción**

Cuando se crea una tarea nueva

Se activa cuando se crea una tarea nueva en un proyecto de Asana. 

Cuando se actualiza una tarea

Se activa cuando se actualiza una tarea existente.

Cuando se crea una subtarea nueva

Se activa cuando se crea una subtarea nueva en una tarea existente.

Cuando se crea una tarea de hito nueva

Se activa cuando se crea una tarea de hito nueva en el proyecto.

Cuando se actualiza una tarea de hito

Se activa cuando se actualiza una tarea de hito en el proyecto.

### Acciones del conector de Asana

**Eventos de acción**

**Descripción**

Crear una tarea

Esta acción crea una tarea nueva en tu proyecto de Asana.

Actualizar una tarea

Esta acción actualiza una tarea existente en tu proyecto de Asana.

Crear una subtarea

Esta acción crea una subtarea nueva y la añade a la tarea principal de tu proyecto de Asana.

Crear una sección en el proyecto

Esta acción crea una sección nueva en tu proyecto de Asana.

Crear una etiqueta 

Esta acción crea una etiqueta nueva en un espacio de trabajo o una organización. 

Crear una historia en una tarea

Esta acción añade una historia nueva a una tarea.

Obtener una tarea

Esta acción devuelve el registro completo de una tarea específica.

## Conector de Asana como paso de acción

1.  En tu cuenta de Kissflow, haz clic en el botón **Crear** del panel de navegación izquierdo y selecciona **Integración** en la lista. Introduce un nombre y una descripción únicos para tu integración y haz clic en **[Crear](/t/60h4akx/creating-an-integration)**. 
    
2.  Configura el paso del activador; por ejemplo, puede ser **Cuando se crea un elemento**. Cada vez que se crea un elemento nuevo en tu Kissflow Process, esto puede desencadenar una acción en la integración.
3.  Después de configurar el paso del activador anterior, haz clic en el botón **Añadir** (![](/migration-assets/xero-connector/1.png)) para configurar el paso de acción de Asana. Busca **Asana** y selecciona un evento de acción disponible. ![](/migration-assets/asana-connector/1.png)
4.  Después de elegir el evento de acción, el siguiente paso es autenticar el conector. Haz clic en **+Añadir una cuenta** e inicia sesión en tu cuenta de Asana. Se te pedirá que uses OAuth para autenticar tu cuenta de Asana.
5.  Después de autenticarte con las credenciales de tu cuenta de Asana, añade los valores correspondientes a los campos del formulario de Asana y haz clic en **Actualizar campos** para ver todos los campos actualizados del formulario de Asana seleccionado. También puedes asignar campos de Asana a los campos del paso del activador o de la acción anterior. Después de ejecutar la acción, los valores de los campos se añadirán automáticamente a los campos de la tarea.![](/migration-assets/asana-connector/2.png)
6.  Después de realizar una configuración correcta, prueba si el paso de acción de Asana está configurado correctamente haciendo clic en el botón **P****robar**. El conector extraerá los datos de los campos de la cuenta de Asana seleccionada y los mostrará como una salida JSON. Al igual que con el paso del activador, se te notificarán los errores de inmediato. Corrígelos antes de volver a probar el paso de acción.

## Configuración de un flujo de trabajo de Asana

Después de crear un flujo de trabajo personalizado con el paso de acción de Asana, activa el botón de alternancia para activar la ejecución de la integración. Cuando la integración esté activa, se creará o actualizará una tarea según el evento del activador que hayas configurado.

Para asegurarte de que tus integraciones estén configuradas y probadas correctamente, consulta nuestra [guía de prácticas recomendadas](/t/p8yzsxn/best-practices-for-using-kissflow-integrations) para obtener consejos sobre cómo crear soluciones sólidas y fáciles de mantener.
