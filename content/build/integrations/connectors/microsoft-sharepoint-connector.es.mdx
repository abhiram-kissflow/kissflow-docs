## ¿Qué es Microsoft SharePoint?

[Microsoft SharePoint](https://www.microsoft.com/en-in/microsoft-365/sharepoint/collaboration) ofrece una solución para compartir y administrar contenido, conocimientos y aplicaciones, con el fin de impulsar el trabajo en equipo y colaborar sin interrupciones en toda la organización de forma segura.

## Acerca de la integración de Microsoft SharePoint con Kissflow

El conector de Microsoft SharePoint de Kissflow Integration te permite conectar y sincronizar tus carpetas y archivos de Microsoft SharePoint con aplicaciones populares de Kissflow. Debes tener una cuenta válida de Microsoft y ser cliente de pago de Kissflow para integrarte con Microsoft SharePoint.

Usa el conector de Microsoft SharePoint para recibir datos de otros flujos o aplicaciones y realizar acciones específicas. El conector de Microsoft SharePoint todavía no admite eventos de activación.

Por ejemplo, después de completar una tarea de Kissflow Process, puedes configurar una integración con el conector de Microsoft SharePoint para guardar archivos adjuntos o archivos en una biblioteca de documentos específica de tu cuenta de Microsoft SharePoint.

### Acciones del conector de Microsoft SharePoint

**Eventos de acción**

**Descripción**

Crear una carpeta

Esta acción crea una carpeta nueva en la biblioteca de documentos de Microsoft SharePoint.

Cargar un archivo

Esta acción carga un archivo o un archivo adjunto de no más de 15 MB en tu biblioteca de documentos de Microsoft SharePoint.

Actualizar valores en columnas personalizadas

Esta acción actualiza los valores de las columnas personalizadas del elemento de tu biblioteca de documentos de Microsoft SharePoint.

## Conector de Microsoft SharePoint como paso de acción

1.  En tu cuenta de Kissflow, haz clic en el botón **Crear** del panel de navegación izquierdo y selecciona **Integración** en la lista. Introduce un nombre y una descripción únicos para tu integración y haz clic en **[Crear](/t/60h4akx/creating-an-integration)**. 
    
2.  Una vez dentro del editor, configura un paso de activación, por ejemplo, un evento de activación de Kissflow Process, **Cuando un elemento completa su flujo de trabajo.**
3.  Después de configurar el paso de activación anterior, haz clic en el botón Agregar (![](/migration-assets/xero-connector/1.png)) junto a él para configurar el paso de acción. Busca **Microsoft** **SharePoint** y selecciona la acción que quieras.![](/migration-assets/microsoft-sharepoint-connector/2.png)   
4.  Después de elegir el evento de acción, autentica el conector. Haz clic en **+Agregar una cuenta** e inicia sesión con tu cuenta de Microsoft. Kissflow usa OAuth de Microsoft para la autenticación.
5.  Después de autenticarte con tu cuenta de Microsoft, añade los valores correspondientes en los campos del formulario de Microsoft SharePoint. Puedes hacer clic en **Actualizar campos** para ver todos los campos actualizados del formulario de Microsoft SharePoint seleccionado.  
    También puedes asignar campos de Microsoft SharePoint desde tu paso de activación o campos de tus pasos de acción anteriores. Estos valores se añadirán automáticamente a los campos del elemento después de ejecutar la acción.
6.  Haz clic en **Siguiente**.![](/migration-assets/microsoft-sharepoint-connector/1.png)
7.  Después de una configuración correcta, comprueba si el paso de acción de Microsoft SharePoint está configurado correctamente haciendo clic en el botón **P****rueba**. El conector extraerá los datos de los campos de la cuenta de Microsoft seleccionada y los mostrará como una salida JSON. Al igual que en el paso de activación, recibirás una notificación inmediata de los errores. Corrígelos antes de volver a probar el paso de acción.

## Configuración de un conector de Microsoft SharePoint

Después de crear un flujo de trabajo personalizado con los pasos de acción de tu conector de Microsoft SharePoint, activa el botón de alternancia para activar la ejecución de la integración. Cuando se produce un evento de activación, se activan las acciones de Microsoft SharePoint.

Para asegurarte de que tus integraciones estén configuradas y probadas correctamente, consulta nuestra [guía de prácticas recomendadas](/t/p8yzsxn/best-practices-for-using-kissflow-integrations) para obtener consejos sobre cómo crear soluciones sólidas y fáciles de mantener.
