The **Files** panel acts as a centralized hub for all attachments within a board. Instead of scrolling through long board forms to find receipts, invoices, or images, you can access them all from a single sidebar. The panel is available within the board workflow, regardless of an item's status.

To open the panel, 

1.  Click **Files**, on the right-hand sidebar inside any board item.
    
2.  The panel will slide out, displaying all attachments grouped by their respective sections.
    

![](/migration-assets/board-files-panel/3.gif)

### Previewing attachments

You can review attachments without leaving the current page or downloading files.

1.  Click directly on any file name (for example, _Screenshot_) within the list.
    
2.  This opens the built-in viewer overlay.
    
3.  From here, you can download, print, open the attachment in a new tab, zoom in or out, rotate, and cycle through multiple attachments using the navigation arrows.
    

### Locating a file in the form

If you need to see the context of an attachment (for example, checking which specific expense row a receipt belongs to):

1.  Hover over the file in the **Files** panel.
    
2.  Click the **View field in form** (![](/migration-assets/board-files-panel/1.png) ) button next to the file name.
    
3.  The main form on the left will automatically scroll and highlight the exact field where that file is attached.
    

![](/migration-assets/board-files-panel/4.gif)

### Searching for a specific attachment

For forms with multiple attachments, use the search bar to quickly find what you need. Search by field name, and the list will instantly filter to show only relevant attachments.

![](/migration-assets/board-files-panel/2.png)
