✓ Basic ✓ Enterprise

Este informe muestra el número de elementos cuyo vencimiento corresponde a cada día. Para acceder a este informe, haz clic en **Reports** en tu tablero. Esto te llevará a la pestaña **Metrics** de forma predeterminada, donde encontrarás el informe resumido de fechas de vencimiento.

![](/migration-assets/due-date-summary-report/3.gif)

Puedes hacer clic en el botón **Interval** (![](/migration-assets/due-date-summary-report/2.svg)) y establecer el intervalo (día, semana, mes, trimestre o año) para ver cuántos elementos vencen dentro de ese intervalo específico. Por ejemplo, si seleccionas **Por día**, el informe muestra cuántos elementos vencen cada día. Si seleccionas el intervalo **Por semana**, es posible que veas **Semana 00** en el eje x. La Semana 00 aparece cuando dos años se superponen y comparten una semana; es decir, cuando el último día de un año y el primer día del año siguiente caen en la misma semana.![](/migration-assets/due-date-summary-report/1.png)También puedes desglosar el informe para identificar rápidamente posibles cuellos de botella en tu tablero. Haz clic en el botón **Break down** (![](/migration-assets/due-date-summary-report/4.svg)) para desglosar el informe por responsable, estado, prioridad, solicitante y campos personalizados. Haz clic en el botón **More options** (![](/migration-assets/due-date-summary-report/5.svg)) para:

-   Copiar el informe como PNG al portapapeles.
-   Descargar el informe como archivo PNG.
-   Imprimir el informe.

Para ver una versión ampliada del informe, haz clic en el botón **Expand** (![](/migration-assets/due-date-summary-report/6.png)). Ahora también tendrás la opción de establecer el período del informe y aplicarle filtros, además de realizar las acciones anteriores. Puedes hacer clic en cada barra del informe para obtener una lista de elementos en el lado derecho de la pantalla. Desde allí, puedes abrir elementos, reasignarlos, cambiar sus estados y prioridades, y eliminarlos.

![](/migration-assets/due-date-summary-report/7.gif)
