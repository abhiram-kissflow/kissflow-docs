## ¿Qué es una lista?

Una lista es un [conjunto de datos](/t/p8h4alk/dataset-and-views-overview) de una sola columna. Los formularios de tus flujos pueden usar los datos de la lista en campos desplegables, listas de verificación, casillas de selección o selección múltiple.

### Ejemplos comunes de listas

-   Departamento.
-   Tipo de permiso.
-   Países.
-   Estados.
-   Tipo de cuenta bancaria.
-   Tipo de pago.![](/migration-assets/list-overview/1.png)

## Dónde usar listas en Kissflow

### [Campo desplegable](/t/35h4azp/dropdown-multi-select-dropdown-checkbox-checklists-and-radio-buttons#dropdown-field)

Úsalo cuando se deba seleccionar una sola opción de una lista de opciones. Por ejemplo, en un proceso de tiempo libre remunerado (PTO), los usuarios pueden seleccionar un solo tipo de permiso, como por enfermedad, vacaciones, personal, etc.

### [Campo de lista de verificación](/t/35h4azp/dropdown-multi-select-dropdown-checkbox-checklists-and-radio-buttons#checklist-field)

Puedes añadir una lista de verificación de elementos que deben completarse como parte de un paso del flujo de trabajo. Por ejemplo, algunos pasos de un proceso de incorporación de empleados pueden incluir listas de verificación, como la creación de cuentas para varias aplicaciones de software por parte del equipo de TI.

### [Campo de casillas de selección o selección múltiple](/t/35h4azp/dropdown-multi-select-dropdown-checkbox-checklists-and-radio-buttons#multi-select-dropdown)

Úsalo cuando quieras permitir que los usuarios seleccionen varios elementos. Por ejemplo, en un proceso de solicitud de viaje, los usuarios pueden seleccionar si necesitan asistencia para vuelos, transporte local o alojamiento.

> **Nota:**
> 
> Puedes añadir hasta 250 elementos a una lista, cada uno con un límite de 500 caracteres.
