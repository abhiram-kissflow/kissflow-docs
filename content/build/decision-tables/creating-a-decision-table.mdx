# Creating a decision table

<PlanBadge plans={["enterprise"]} />

You create decision tables inside an app in the development environment.

## Create a decision table

1. Open your app in the app builder.
2. In the left panel, open the **Logic** tab and click the **Create** button.
3. Enter a name and an optional description, then click **Create**.

![Animation of creating a decision table: clicking the Create button in the app builder's left panel, entering a name and description, and clicking Create](/migration-assets/creating-a-decision-table/4.gif)
{/* TODO: verify screenshot matches current product UI */}

A new, empty decision table opens. Add [conditions and results](/docs/build/decision-tables/conditions-and-results) to start defining your logic.

## Rename a decision table

To rename a decision table, click the **More options** button at the top right and select **Rename**. You can change the description the same way.

{/* TODO: screenshot — More options menu */}

## Find a decision table

Your decision tables are listed under the **Logic** tab in the app builder's left panel.

![Logic tab in the app builder's left panel listing the app's decision tables](/migration-assets/creating-a-decision-table/3.png)
{/* TODO: verify screenshot matches current product UI */}

## Limits

A decision table supports up to **10 conditions**, **5 results**, and **50,000 rows**.
