Los [Superadministradores, administradores de IAM y administradores de usuarios](/t/35h4ak036h4gg/user-roles-permissions) pueden exportar los detalles de algunos o de todos los usuarios de una cuenta de Kissflow.

## Exportar datos de usuarios

Para exportar tus datos de usuario, sigue los pasos que se indican a continuación:

Inicia sesión en Kissflow. Haz clic en tu foto de perfil > **Administración de la cuenta** > **Gestión de usuarios**.![](/migration-assets/exporting-user-data/1.png)    2.  Haz clic en el botón **Exportar datos de usuarios** (![](/migration-assets/exporting-user-data/2.png)) en la tabla de gestión de usuarios.
3.  Selecciona el botón de opción correspondiente en el menú desplegable para exportar los datos de usuario necesarios.
    -   **Datos filtrados**: exporta los detalles únicamente de los registros de usuarios filtrados. 
    -   **Todos los datos**: exporta los detalles de todos los usuarios de tu cuenta de Kissflow.
        
        ```
        Note:
        When you select Filtered data, all the data fields hidden from your page are
        filtered out, only visible data is available in your exported file.
        ```
        
4.  Selecciona el tipo de archivo: CSV o JSON. Haz clic en **Exportar**.

Cuando hagas clic en Exportar, recibirás un correo electrónico con un enlace para descargar el archivo exportado. También puedes descargar el archivo exportado accediendo a la [tarjeta de notificación](/t/x2h4azyy7h4gj/managing-notifications) correspondiente.

> **Nota:**  
> Si utilizas el botón **Mostrar/ocultar campos** para ocultar algún campo, solo podrás exportar la página actual y no todos los datos.
