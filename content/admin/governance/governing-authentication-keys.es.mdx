Planes de Kissflow:

✕ Basic ✓ Enterprise

Gobernar las claves de autenticación es crucial por varias razones. En primer lugar, ayuda a mantener la seguridad y la integridad de tu cuenta de Kissflow y sus datos. Como [Superadministrador o administrador de IAM](/t/35h4ak0/user-roles-and-permissions), puedes controlar quién tiene acceso a tus API e integraciones mediante la gestión y el gobierno de estas claves. Esto minimiza el riesgo de accesos no autorizados o filtraciones de datos. Una gobernanza adecuada de las claves de autenticación también permite realizar un mejor seguimiento y auditoría de las API. Puedes consultar todas las claves de autenticación (claves de acceso y de API) de tu cuenta en una sola página. Se muestran detalles como la fecha de vencimiento de las claves, el tiempo de actividad, la fecha y el autor de creación, la última fecha de acceso y su estado, para que puedas tener una visibilidad adecuada.![](/migration-assets/governing-authentication-keys/1.png)
